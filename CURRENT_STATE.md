# Current State

## Реализовано

Проект Telecom Manager создан как FastAPI веб-приложение с PostgreSQL, SQLAlchemy 2.0, Alembic, Jinja2, Bootstrap 5, HTMX и Alpine.js.

Реализованы:

- базовая структура проекта;
- конфигурация через env;
- логирование;
- SQLAlchemy engine и Session;
- Alembic окружение;
- первая миграция;
- seed складов, материалов и администратора;
- авторизация;
- login/logout;
- Dashboard;
- 404 страница;
- левое меню и верхняя панель пользователя;
- модуль материалов;
- модуль подключений.

## Известные ограничения

- На текущей машине не установлен Docker и PostgreSQL CLI, поэтому миграции невозможно применить локально без внешней БД.
- В `.env.example` PostgreSQL host установлен как `postgres`, что рассчитано на docker compose сеть.
- Для локального запуска без Docker нужно установить PostgreSQL и указать `POSTGRES_HOST=localhost`.
- Авторизация реализована через подписанную cookie-сессию без отдельной таблицы сессий.
- Удаление подключений доступно только ADMIN, остальные права будут расширяться позже.
- Справочники складов и материалов создаются миграцией seed-данных.
